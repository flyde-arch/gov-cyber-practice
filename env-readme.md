# 环境说明文档 env-readme.md
1. 操作系统版本：CentOS Stream 10
2. 虚拟机配置：VMware Workstation 搭建，2核/4G内存/40磁盘（看你创建虚拟机时的配置填写，比如2核4G内存 60G磁盘）
3. 学习周期：12天
4. 整体学习目标：掌握Linux基础命令、网络配置、服务器运维基础，完成政企安全相关实操练习
# 实训目录用途说明
1. firewall：存放防火墙规则、黑白名单、防火墙运行日志与配置文件
2. attack-detect：存放入侵告警日志、恶意流量抓包、攻击行为排查记录
3. access-control：存放账号权限配置、SSH登录白名单、系统审计日志
4. incident-response：存放安全事件处置流程、入侵溯源报告、故障修复文档
